# Kickstarter-Tableau-Project
This project analyzes Kickstarter campaigns to understand what drives project success.


## Overview

- This project analyzes Kickstarter campaigns to understand what drives project success.
  
- The dataset includes thousands of projects from various categories, focusing on funding performance, success rates, and audience engagement.
  
- The analysis was visualized using Tableau.


## Objective

To identify patterns and key factors that make a Kickstarter campaign successful and highlight which categories perform best or struggle to attract backers.


## Key Insights

- Film & Video and Design categories achieved the highest funding success.

- Dance and Photography received the lowest average funding, showing limited backer interest.

- Creative and visually appealing projects attract stronger audience support.


## Recommendations

- Prioritize visual categories like Film & Video and Design, which perform best in funding and success rate.

- Enhance promotion for weaker fields such as Dance and Photography.

- Focus on creative quality. strong, well-presented ideas draw more backers.


## Tools Used

- Tableau for data visualization
  
- Excel / CSV for initial data cleaning: Calculated Fields for data correction (e.g., merging Film & Video duplicates)



## Deliverables

- Tableau Dashboard (.twbx)
  
- Presentation Slides (Recommendations & Insights)
